## 📗 상세 내용

- **사용 대상** : 반려 동물을 키우는 1인 가구
- **선정 이유** : 혼자 사는 사람들이 집에 혼자 있을 반려 동물을 캠을 통해 보고 반려 동물의 상태를 확인하게하여 안심하게 하기 위해
- **기능** : 카메라가 반려 동물을 인식해 움직이며 반려 동물을 관찰
- **기대 효과** : 반려 동물을 키우는 사람들이 안심하고 직장이나 외출을 할 수 있게 된다.

- 파이어베이스 활용(스토리지, 로그인 인증, 소셜 로그인)
